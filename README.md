## 3. 화면구성 (스플래쉬 & 로그인 제외)
### 1) 홈 (대시보드)
  - 인사말, 카드(대시보드), 거래(지출) 내역
### 2) 지출 분석
  - 차트, 거래(지출) 내역 + 정렬
  - 원호 애니메이션
### 3) 예산 현황
  - Bar 차트 (지출, 예산)
  - 총 지출 (이번 달)
  - 지출 내역
### 4) 카드 관리
  - 카드 이미지(ViewPager)
  - 카드 이름, 카드 번호, 종류
### 5) 입력 화면
  - 지출 입력
  - 카드 입력
